# NVIDIA Financial Performance Dashboard (2018–2024)

This project analyzes NVIDIA’s financial performance from 2018 to 2024 using an interactive Tableau dashboard. It highlights key financial metrics, long‑term trends, and the rapid growth of the company’s data center segment.

## Project Overview
The dashboard provides an executive‑level view of NVIDIA’s financial trajectory. It includes:
- Key Performance Indicators (KPIs)
- Revenue trends
- Profitability trends
- Segment performance (Data Center vs Gaming)
- Earnings per share (EPS)
- Operating cash flow
- Year‑over‑year (YoY) changes

The goal of this project was to practice data cleaning, metric engineering, visualization design, and business storytelling.

## Data Preparation
The dataset was manually cleaned and structured in Excel/CSV format. Key steps included:
- Standardizing financial values
- Engineering YoY metrics for revenue, net income, data center revenue, gaming revenue, and cash flow
- Validating EPS and operating cash flow values
- Preparing the dataset for Tableau import

## Dashboard Features
1. KPI Row  
   - Revenue (FY2024)  
   - Net Income (FY2024)  
   - Data Center Revenue (FY2024)  
   - EPS (FY2024)

2. Revenue Trend  
   Line chart showing NVIDIA’s revenue growth from 2018–2024.

3. Profitability Trend  
   Net income trend with clear visualization of major jumps and declines.

4. Segment Revenue  
   Comparison of Data Center vs Gaming revenue, highlighting NVIDIA’s shift toward AI infrastructure.

## Key Insights
- Revenue grew from $9.7B in 2018 to $60.9B in 2024.
- Net income increased more than 28× over the same period.
- Data Center revenue grew from $1.9B (2018) to $47.4B (2024), a 15× increase.
- Gaming revenue remained strong but was overtaken by Data Center as the dominant segment.
- EPS rose significantly, reflecting strong profitability and operational efficiency.
